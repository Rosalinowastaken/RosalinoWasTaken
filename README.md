Hey there

Soy Rosalino, desarrollador full stack basado en Madrid, España.

Trabajo principalmente en backend y arquitectura segura — Java, Spring Boot, FastAPI, y JWT/OAuth2 para autenticación y control de acceso.

En frontend uso React, Angular y TypeScript, y para desplegar todo, Docker, Nginx y Linux.

Actualmente estoy cerrando mi Máster en Ciberseguridad e IA, trabajando en Valhalla SOC: un SOC local con detección de amenazas, honeypots y un copiloto de IA para triage.
